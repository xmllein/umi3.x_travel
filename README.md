## 项目运行

- `yarn start` 启动项目，带有热更新功能，`mock`数据
- `yarn start:no-mock` 启动项目，带有热更新功能，不带`mock`数据（启动关联后台服务）
- `yarn build` 打包项目，打包后的文件在`dist`目录下
